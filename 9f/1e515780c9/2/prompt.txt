# Plan: Update empty state heading + animate button on press

## Context
The empty list screen shows "No recipes yet" with a large + button. The user wants to change the heading to "Add your first recipe" and add a smooth scale-down + fade-out animation on the + button when it's pressed, so the popover appears over empty space.

## Changes

### 1. Update translations
- **`platform/i18n/translations/en.json`**: Change `emptyState.title` from `"No recipes yet"` to `"Add your first recipe"`
- **`platform/i18n/translations/pl.json`**: Change `emptyState.title` from `"Brak przepisów"` to `"Dodaj swój pierwszy przepis"`

### 2. Animate the EmptyState when popover opens
- **`features/photos/components/EmptyState.tsx`**:
  - Add a `hiding` boolean prop driven by `popoverVisible || isImporting` from the parent
  - Use `usePopoverTransition` (or equivalent spring animation) to animate scale (1→0.8) and opacity (1→0) when `hiding` is true
  - Wrap the button (or entire container) in `Animated.View` with the animated style
  - Reuse the existing spring config pattern (`damping: 40, stiffness: 200`)

- **`app/index.tsx`**: Pass `shouldHideSearchBar` (already computed as `popoverVisible || isImporting`) to EmptyState as `hiding`
