# Commands

## Development

```bash
npm start                    # Start Metro bundler (JS-only changes)
npx expo run:ios            # Build and run on iOS Simulator
npm run ios                 # Build for physical iOS device (handles provisioning)
npm run android             # Build and run on Android
npm run web                 # Start web version
```

## Quality

```bash
npm run lint                # Run ESLint
```

## Utilities

```bash
npm run test-classifier     # Test text classifier CLI (npx tsx lib/text-classifier/cli.ts)
```
